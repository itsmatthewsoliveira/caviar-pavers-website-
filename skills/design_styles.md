# Design Styles & Visual Direction

## Core Aesthetic: "Structural Elegance"
- **Reference:** Directly modeled on the "Luzen" layout.
- **Vibe:** Professional, Architectural, "Upright".

## Layout Structure (Strict Adherence to Reference)
1.  **Header:**
    - Logo (Caviar Pavers).
    - Navigation.
    - **CRITICAL:** **Lead Capture Form** embedded directly in the hero/header area.
2.  **Hero Section:**
    - Full-height premium image (Paver/Pool).
    - Headline: "ELEGANCE MEETS DURABILITY".
    - Subtext + Call to Action.
3.  **Section 01 (Narrative):**
    - Split layout (Text Left / Images Right).
    - Start of "Services" breakdown.
4.  **Section 02 (Services Breakdown):**
    - **Hardscaping:** Patios, Pools, Walkways.
    - **Softscaping/Features:** Artificial Turf, Screen Enclosures.
    - *Style:* Clean, elegant lists or refined cards (NOT boxy).
5.  **Section 03 (Process/Stats):** "Our Success driven by..." equivalent.
6.  **Footer:** Large functional footer with "Contact" focus.

## Visual Elements
- **Typography:** Montserrat (Bold/ExtraBold for headers).
- **Colors:** Deep Blue backgrounds for contrast sections. Gold lines/dividers.
- **Forms:** Clean, minimalist input fields (line-only or slight background).
- **Images:** High-resolution, professional, aspirational.

## Component Specifics
- **Lead Form:** "Get a Free Estimate". Fields: Name, Phone, Email, Service Type.
- **Buttons:** Sharp or slightly rounded. Blue with Gold text/border or vice versa.
